# mat-status-tracker
Tracks mats status on discor d


# Usage:

# Step 1
Confirm the file is safe (wow!)

# Step 2
Download python (wow!)

# Step 3
Go in command prompt and run this:
py -m pip install "discord"
This will download the libary needed to run this

# sTEP 5
Go to https://discord.com/developers/applications
Make a new application
Once you've done that go to the Bot tab and turn on Presence Intent, Server Members Intent, and Message Content Intent. 

# Step 6/7
If you don't have your bot token, get it by going to the Bot tab in the discord dev portal and clicking reset your token. Get your token in your copy and paste.
Download matstatusupd.py, then open the file in notepad, scroll down untl you see bot.run
Where the code says Token, replace Token with your discord bot token. Keep the ""
It should look like this: bot.run("{your token}") where {your token} is well your discord bot token

# Step 8
Open command prompt and type
cd downloads
then
py -3 matstatusupd.py
